Project Dscription
-------------------------------------------------------
The purpose of this project is for a user to be able to manage and store Nintendo Ds Game data. This application allows the user to add, view, update, delete
and search Nintendo Ds Games by genre. This System is called the Nintento Ds Game Management System or NDS-GMS for short, and it takes advantage of using
CRUD operations and a Custom Action. 

Input/Output
-------------------------------------------------------
The expected input for the NDS-GMS is for the user to choose an menu item numbered from 1 to 6 for how they want to interact with a Ds Game. If option 1 is selected
which is to add a ds game, the user has the option to add a ds game to the system by either manually or by loading a text file. If the user chooses to add a
game manually, the system will prompt the user to enter the required data for each DS Game data attribute such as the title, developer, genre, release year,
multiplayer status and rating. Once all data is entered, the output of the system will display to the user that the game was stored successfully or if there 
was invalid data. If option 2 is selected which is to view all ds games, the expected output will display the current games within the system in the correct format.
If option 3 is selected which is to update a ds game, the expected input would be the user to enter the title of the ds game they want to update. If entered 
correctly, the user will then be taken to a sub-menu numbered from 1 to 7 where they can choose the desired data they want to update. Whichever option they choose 
the expected output will displayed that the desired data has been updated. The output when viewing all ds games in the system will show the users updated ds game. 
If option 4 is selected which is to delete a ds game, the user will be prompted to input the title of the desired Ds Game they want to delete. If successfully
deleted the system will output will display the ds game that was removed. If not successful, the expected output will display to the user that there is no game 
with that title or there was an invalid input. If Option 6 is selected which is a custom action that allows the user to search ds games by genre, then the expected
input from the user is for them to enter the desired genre they want to search for. If there are Ds Games with the desired matching genre, then the expected output
will be a display of all current Ds Games in the system with the same desired genre from the user. Finally, if option 6 is selected, then this allows the user to
completely exit the program, and the expected output is that the user succesfully exited the NDS-GMS.
