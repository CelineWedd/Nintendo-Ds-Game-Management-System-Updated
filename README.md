Project Dscription
-------------------------------------------------------
The purpose of this project is for a user to be able to manage and store Nintendo Ds Game data. This application allows the user to add, view, update, delete
and search Nintendo Ds Games by genre. This System is called the Nintento Ds Game Management System or NDS-GMS for short, and it takes advantage of using
CRUD operations and a Custom Action. 

Input/Output
-------------------------------------------------------
The expected input for the NDS-GMS is for the user to choose an menu item for how they want to interact with a Ds Game. If Add Ds Game is selected,
the user has the option to add a ds game to the system by either manually or by loading a text file. If the user chooses to add a
game manually, the system will prompt the user to enter the required data for each DS Game data attribute such as the title, developer, genre, release year,
multiplayer status and rating. Once all data is entered, the output of the system will display to the user that the game was stored successfully or if there 
was invalid data. If View Ds is selected, the expected output will display/refresh the current games within the system in the correct format.
If Update Ds Game is selected, the expected input would be the user to select the ds game from the table view list that they want to update. If selected 
correctly, the user will then be taken to a sub-menu where they can choose the desired data they want to update. Whichever option they choose 
the expected output will display that the desired data has been updated. The output when viewing all ds games in the system will show the users updated ds game. 
If Remove Ds Game is selected, the user will be prompted to select the desired Ds Game they want to remove. If successfully
removed the system will display the ds game that was removed. If not successful, the expected output will display to the user that there is no game 
with that title or there was an invalid input. If Search By Genre is selected which is a custom action that allows the user to search ds games by genre, then the expected input from the user is for them to enter the desired genre they want to search for. If there are Ds Games with the desired matching genre, then the expected output will be a display of all current Ds Games in the system with the same desired genre from the user. Finally, if Exit is selected, then this allows the user to completely exit the program, and the expected output is that the user succesfully exited the NDS-GMS.
